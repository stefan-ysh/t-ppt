## 🎯 立即使用指南

### 快速启动开发环境

```bash
# 1. 启动开发工具菜单
./dev-tools.sh

# 2. 或使用 npm 脚本
npm run tools

# 3. 快速开发模式
npm run dev:home    # 启动首页预览
```

### 图片优化工作流

```bash
# 1. 分析项目图片
npm run optimize:images

# 2. 获取WebP转换指南  
npm run webp:guide

# 3. 分析WebP转换潜力
npm run webp:analyze
```

### 构建和部署

```bash
# 1. 增强构建 (推荐)
npm run build:enhanced

# 2. 性能分析
npm run performance

# 3. 预览构建结果
npm run preview
```
